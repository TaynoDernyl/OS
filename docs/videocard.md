| Функция                          | Описание                                                                                                  |
| -------------------------------- | --------------------------------------------------------------------------------------------------------- |
| vga_init()                       | Инициализирует видеокарту. Все пиксели устанавливаются в чёрный цвет (значение `0x00`).                   |
| vga_clear()                      | Очищает консольный экран, удаляя все предыдущие отрисованные символы.                                     |
| vga_set_pixel(x, y, color)       | Устанавливает пиксель по координатам **(x, y)** в указанный цвет. Цвет задаётся байтом `uint8_t` (0–255). |
| vga_render()                     | Выполняет отрисовку текущего состояния видеопамяти на экран, делая изменения видимыми пользователю.       |
